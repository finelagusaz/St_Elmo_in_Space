# 真空のセントエルモ

あなたはどこかの資源惑星を護衛する宇宙空母のクルーです。

他のクルーと話して見識を深めたり、AI「スクルド」と話して親密度をあげてみましょう。

もしかすると、彼女の変わった一面が見られるかもしれません。
